Keyboard shortcuts help you work faster by providing quick access to common operations without reaching for the mouse.

## All Shortcuts

<Tip>
Keyboard shortcuts work in both 2D and 3D view modes. Use them to quickly manipulate bins while designing your layout.
</Tip>

### Bin Operations

These shortcuts work when you have a bin selected:

| Key | Action | Description |
|-----|--------|-------------|
| `R` | Rotate bin | Swap width and depth (e.g., 1x4 becomes 4x1) |
| `Delete` | Remove bin | Delete the selected bin from the grid |
| `Backspace` | Remove bin | Alternative key to delete the selected bin |
| `Escape` | Cancel operation | Cancel placing, dragging, or resizing a bin |

<Tip>
Press `R` on non-square bins to quickly rotate them. This is faster than manually adjusting width and depth values.
</Tip>

### Grid Navigation

Control the 2D grid view with these shortcuts:

| Key | Action | Description |
|-----|--------|-------------|
| `Shift + Left Click` | Pan grid | Hold Shift and drag to move the viewport |
| `Middle Mouse` | Pan grid | Hold middle mouse button and drag to pan |
| `Scroll Wheel` | Zoom in/out | Zoom between 0.2x and 5x magnification |

### History

Undo and redo operations to experiment freely:

| Key | Action | Description |
|-----|--------|-------------|
| `Ctrl+Z` | Undo | Revert the last change |
| `Ctrl+Shift+Z` | Redo | Reapply an undone change |

<Tip>
Gridfinity Builder tracks your complete history, so feel free to experiment. You can always undo back to a previous state.
</Tip>

## Workflow Tips

1. **Quick Bin Rotation**: Select a bin and press `R` repeatedly to try different orientations
2. **Mass Delete**: Use `Delete` key to quickly remove unwanted bins one by one
3. **Explore Freely**: Use `Ctrl+Z` to undo, then try different layouts without fear
4. **Smooth Navigation**: Combine `Scroll Wheel` zoom with `Middle Mouse` pan for precise positioning

## Platform Notes

- **Windows/Linux**: All shortcuts work as documented
- **macOS**: Use `Cmd` instead of `Ctrl` for undo/redo
- **Touchpad**: Two-finger scroll for zoom, three-finger drag for pan (if supported)

<Tip>
Practice using keyboard shortcuts while designing. After a few sessions, they'll become second nature and significantly speed up your workflow.
</Tip>
