## How to Use Templates

1. **Load a template** from the sidebar template selector
2. **Preview the layout** in 2D and 3D view
3. **Customize bins** - change sizes, add features, adjust dividers
4. **Add more bins** using presets or the bin configurator
5. **Export as 3MF** when ready to print

<Warning>
Loading a template will **replace** your current layout. Save your work first if you want to keep it.
</Warning>

## Customizing Templates

After loading a template, common modifications include:

### Adjusting Bin Heights

Change bin heights to fit your items:

- **Shallow (2-3u)**: Small parts, screws, electronics components
- **Medium (4-5u)**: Hand tools, cables, office supplies  
- **Tall (6-7u)**: Screwdrivers, long tools, bottles

### Adding Dividers

Dividers help separate items within a bin:

- **X dividers**: Vertical walls running left-to-right
- **Y dividers**: Vertical walls running front-to-back
- **Combinations**: Create grid compartments (e.g., 3x3 = 9 sections)

<Tip>
Use dividers for small parts that would otherwise mix together. The Socket Tray preset uses 3x3 dividers to create 9 compartments.
</Tip>

### Adding Features

- **Stacking lip**: Enables stacking bins on top of each other
- **Label shelf**: 45° angled shelf for paper labels
- **Magnets**: 6mm holes for holding bins to steel surfaces
- **Screws**: M3 holes for permanent mounting

## Creating Custom Templates

While you can't save custom templates in the app yet, you can:

1. **Save layouts** to browser localStorage
2. **Share via URL** - Copy the share link to bookmark your layout
3. **Export and reimport** - Use the save/load feature for frequent layouts

See the [Sharing guide](/guides/sharing) for details.

## When to Use Each Template

| Template | Best For | Avoid If |
|----------|----------|----------|
| **IKEA Alex** | Desk organization, office supplies | Electronics (too shallow) |
| **Electronics Bench** | Hobbyist projects, component storage | Large tools (grid too small) |
| **Tool Wall** | Workshop walls, shallow drawers | Deep storage needs |
| **Server Rack 1U** | Network closets, datacenters | Non-rack environments |

<Tip>
Start with the template closest to your use case, then customize. It's faster than designing from scratch.
</Tip>
