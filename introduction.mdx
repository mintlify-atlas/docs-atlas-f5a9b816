## What is Gridfinity Builder?

Gridfinity Builder is a browser-based parametric CAD tool that lets you design, preview, and export 3D-printable Gridfinity storage layouts. Built for makers, 3D printing enthusiasts, and anyone who wants to organize their workspace with the [Gridfinity modular storage system](https://gridfinity.xyz).

<Note>
Gridfinity is an open-source modular storage system created by [Zack Freedman](https://www.youtube.com/@ZackFreedman). This tool helps you design custom bin layouts and export them as 3MF files ready for slicing.
</Note>

## Why Gridfinity Builder?

<Steps>
  <Step title="Design faster">
    Interactive 2D grid with drag-and-drop placement, resize handles, and rotation controls. No CAD experience required.
  </Step>
  <Step title="Preview accurately">
    Real-time 3D rendering powered by Three.js and Manifold CSG engine. See exactly what you'll print.
  </Step>
  <Step title="Print immediately">
    Export watertight, manifold 3MF files that open directly in your slicer. No cleanup required.
  </Step>
</Steps>

## Supported Printers

Gridfinity Builder includes presets for popular 3D printers:

- **Bambu Lab**: A1, A1 Mini, P1S, X1C, X1E, H2D (6×6 grid, 252×252mm)
- **19" Server Rack**: 10×8 grid (420×336mm)
- **Custom**: Any size baseplate

All exported files work with any FDM 3D printer and slicer software.
